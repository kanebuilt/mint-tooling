# Mint

> Develop TurboWarp extensions using bundling

---

## Overview

The Mint tooling is a new bundling toolchain for TurboWarp extensions. It allows you to write your extension using modern JavaScript features and bundle it into a single file that can be loaded by TurboWarp. The toolchain is designed to be easy to use and flexible, allowing you to customize the build process as needed.

## Key Features

- **Simple & modern:** Write your extension using modern JavaScript syntax and features. No huge dependencies needed!
- **Bundling:** Bundle your extension into a single file that can be loaded by TurboWarp. No more monolithic source files!
